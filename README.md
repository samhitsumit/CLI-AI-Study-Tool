# 📚 AI Study Tool

Hi! My name is **Samhit** and I am 12 years old. I made this project
because I wanted studying to be easier and more fun.

This AI Study Tool can help students learn by making questions,
flashcards, summaries, and giving feedback using AI.

## ✨ Features

-   📝 Summarize your notes into short bullet points.
-   📖 Generate flashcards from notes or images.
-   ❓ Create multiple choice questions.
-   ✍️ Create open-ended questions.
-   🤖 Chat with an AI tutor and ask any study questions.
-   📊 Track your quiz scores and see your progress.
-   💾 Save questions you got wrong so you can practise them later.
-   📷 Read notes from images using OCR.
-   🧠 Generate similar questions for extra practice.
-   📝 Get feedback on questions you answered wrongly.

## 🛠️ What I Used

-   Python
-   Ollama
-   EasyOCR
-   Rich
-   Questionary
-   Matplotlib

## 🚀 How to Run

1.  Install Python.
2.  Install all the required Python libraries.
3.  Make sure Ollama is installed and running.
4.  Download the Gemma model.
5.  Run:

``` bash
python main.py
```

## 📂 Files

-   `main.py` - Runs the whole program.
-   `funcs.py` - Contains all the AI functions and study tools.
-   `data.json` - Stores quiz history, scores, and saved questions.
